# Cryptocurrency Price Ticker

A real-time cryptocurrency price ticker application built with Python and Tkinter. Fetches live price data from the CoinGecko API.

## Features

- Real-time price updates for multiple cryptocurrencies
- 24-hour price change indicators (green/red)
- Market cap display
- Multiple fiat currency support (USD, EUR, GBP, JPY, CNY, KRW)
- Add/remove cryptocurrencies dynamically
- Automatic refresh every 30 seconds
- Manual refresh button
- Scrollable interface
- Rate limiting and error handling

## Requirements

- Python 3.7+
- pip (Python package manager)
